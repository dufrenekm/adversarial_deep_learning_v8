To set up:
Create a Python virtual environment with:
```console
python3 -m venv deep_learning_env
```

Activate with
```console
source deep_learning_env/bin/activate
```

Install requirements
```console
python3 -m pip install -r requirements.txt
```

Test the model
```console
python3 test.py
```
